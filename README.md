# Juego del Ahorcado en Python

## Estudiante
Jorge Calderón

## Objetivo del sistema
Desarrollar un juego de Ahorcado funcional en Python que aplique estructuras 
lógicas (condicionales y repetitivas), manejo de listas/tuplas y módulos 
externos, como parte del Proyecto Integrador de Lógica de Programación.

## Descripción de funcionalidades
- Selección aleatoria de palabra mediante el módulo `random`
- Control de intentos del jugador
- Visualización del progreso de la palabra (letras adivinadas/ocultas)
- Validación de letras repetidas
- Mensajes de victoria/derrota al finalizar la partida

## Estructura del repositorio
- `/src` → código fuente en Python
- `/diagramas` → diagrama de flujo (.rap, .png) y diagrama de arquitectura (.png)
- `/documentacion` → documento del proyecto integrador
- Videos explicativos (Parte 1, Parte 2 y Proyecto Integrador)
